User Story #001: As a developer, I want to create a well-structured prompt through a chat interface so that I can quickly generate markdown documentation for my prompts.

Title: Create Prompt via Chat Interface

As a: Developer/Engineer
I want: To interact with a chat-based interface that guides me through creating a structured prompt
So that: I can quickly generate a properly formatted markdown file containing my prompt

Acceptance Criteria:
- The system should provide a welcoming chat interface when I start a new prompt project
- The system should ask clarifying questions about the purpose and use case of my prompt
- The system should validate my inputs and request clarification when needed
- The system should provide real-time preview of the prompt structure as I build it
- The system should generate a complete markdown file when I'm satisfied with the prompt
- The system should allow me to export or save the markdown file to my preferred location

Technical Notes:
- Should integrate with the OpenClaw ecosystem
- Should support template selection and customization
- Should maintain conversation context throughout the prompt building process
- Should provide error handling for invalid inputs
